## 🔧 Customization

### Change Messages

Edit `src/services/AIService.js`:

```javascript
getHighRiskAlert = (riskScore, location) => {
  return {
    message: 'YOUR_MESSAGE_HERE',  // ← Modify this
    severity: 'high',
    icon: '⚠️',
    shouldShow: true,
    color: COLORS.danger,
  };
};
```

### Change Thresholds

Edit `src/config.js`:

```javascript
export const RISK_CONFIG = {
  nightTime: { hour: 21 },     // ← Change to 20 for 8PM
  morningTime: { hour: 6 },    // ← Change as needed
  noMovementThreshold: 30,     // ← Change sensitivity (seconds)
  noiseSpikeThreshold: 70,     // ← Change threshold (dB)
  sparseAreaThreshold: 5000,   // ← Change sensitivity (meters)
};
```

### Change Colors

Edit `src/components/AIPredictionBanner.js` OR `src/config.js`:

```javascript
export const COLORS = {
  danger: '#EF4444',    // ← Red for critical
  warning: '#F59E0B',   // ← Yellow for medium
  success: '#10B981',   // ← Green for safe
  primary: '#6366F1',   // ← Blue for info
};
```

### Change Animation Speed

Edit `src/components/AIPredictionBanner.js`:

```javascript
Animated.timing(slideAnim, {
  toValue: 0,
  duration: 400,  // ← Change from 400ms to desired duration
  useNativeDriver: true,
})
```

## 🐛 Common Issues & Solutions

### Banner never appears

**Check**:
1. Risk score > 50 (see gauge)
2. `shouldShow` is true (check console)
3. Routes are loading properly

**Solution**: See troubleshooting in `AI_TESTING_GUIDE.md`

### Animation stutters

**Check**:
1. `useNativeDriver: true` is set
2. No heavy computations in banner

**Solution**: Already optimized - shouldn't occur

### Text gets cut off

**Check**:
1. Screen size
2. Font size correct

**Solution**: Adjust `SPACING` or font size in styles

### AutoHide not working

**Check**:
1. Risk didn't actually drop below 50
2. `slideOut()` function

**Solution**: Verify `shouldShow` changes to false
